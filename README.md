# Dispatcher

A terminal multiplexer desktop app built with Tauri, React, and xterm.js.

## Tech Stack

- **Frontend:** React, TypeScript, xterm.js, Zustand
- **Backend:** Tauri (Rust), portable-pty
- **Build:** Vite

## Development

```bash
npm install
npm run tauri dev
```

## Building

```bash
npm run tauri build
```
